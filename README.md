# thumbvids
I have always wondered how a site like Canva was built so I decided to do something like it. While this is not an exact Canva replica, I made an attempt to build my own complete site with a couple of templates to get anyone started. The link to access the site is https://thumbvids.herokuapp.com . Please check it out and drop some comments on the Portfolio page for me to see. The page will eventually lead to my portfolio website which I have not found time to complete yet. This is built in Vanilla Javascript and Fabric.js using HTML5 Canvas. The entire app is just an events app listening to your Clicks and all. Well, there are quite a few other DOM manipulations going on which you will figure out :). You can edit Templates or even create your own designs from scratch. The site is designed to create graphics for Youtube Thumbnails with dimension (1280 x 720). Edit Templates, Create Graphics and Download them on your local device. You can even sign up and keep a record of your designs on the app.
